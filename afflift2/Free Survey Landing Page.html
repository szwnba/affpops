Free Survey Landing Page


My chat landing page is getting pretty popular so I thought you might want another landing page to split test it against. I created my own survey type landing page to see how it would perform compared to the chat LP and it is doing pretty well.



I am using push ads to promote the landing page. I decided to create it with Bootstrap because I really like it. Some people aren't fans of Bootstrap, but I've been using it since it was released and it seems to convert well

So, it's very basic. In fact, here's the code:

I have included the files as an attachment below so you can download the complete LP (images included).

The core of the landing page is the question divs.

Code: 
<div class="question">
    <h3>Did you receive a message from her? ☝️</h3>
    <a href="#" class="btn btn-success btn-block answer">Yes</a>
    <a href="#" class="btn btn-danger btn-block answer">No</a>                       
</div>

You can add as many questions as you like and the JavaScript will go through them each when one of the buttons is clicked. The data isn't saved (what is chosen), but that doesn't matter really in my opinion. I recommend only using a few questions.

The last section, results, is where your final call-to-action is:

Code: 
<div class="results" style="display: none">
    <h3>Amahle is waiting!</h3>
    <a href="https://v.newlyclicked.com/click" class="btn btn-success btn-block">
    Click <strong>HERE</strong> to Connect and Chat 
    </a>
    <hr>
    <p>
    (it takes about 1 minute to register and connect)
    </p>
</div>

As I said, this LP is performing pretty well for me. This morning, it has over a 40% CTR on PropellerAds



Of course, feel free to change it how you see fit. Enjoy!
