In my Triple Threat thread, I talked about how I am using back button and popunder scripts to help monetize my landing pages. Well, I decided to combine them into a single script that you and I can use on any landing page really easily 

I call it BackUnder 

BackUnder Script

JavaScript:
<script>
var back = 'https://yahoo.com';
var under = 'https://google.com';</script><script src="https://backunder.com/script.js"></script>

All you need to do to add Back Button or Popunder to your landing page is copy/paste this code onto the landing page (I recommend the inside the <head></head>) and change the links you want to send the person to.

In the example, someone that presses the back button in their browser will be taken to yahoo.com and after someone clicks any link on your landing page, google.com will load as a popunder and your main call-to-action link will be loaded in a new tab (which becomes the focus when clicked).

You can remove either variable to remove that functionality from your landing page. For example, if I only wanted to use the back button feature and not the popunder feature, I would simply add this code to my landing page:

Only Back Button

JavaScript:
<script>
var back = 'https://yahoo.com';</script><script src="https://backunder.com/script.js"></script>

Or, if I only wanted to use the popunder and remove back button redirect:

Only Popunder

JavaScript:
<script>
var under = 'https://google.com';</script><script src="https://backunder.com/script.js"></script>

If your landing page has multiple links or uses any sort of JavaScript links (like we use with our Free Survey Landing Page), then you may want to define which specific link triggers the popunder. You can do that by adding a cta variable and then giving that link the class you define. For example:

Popunder with Specific CTA

JavaScript:
<script>
var cta = 'mainButton';
var back = 'https://yahoo.com';
var under = 'https://google.com';</script><script src="https://backunder.com/script.js"></script>

In this case, the link that is my main call-to-action would need the class mainButton:

HTML:
<a href="https://yourtrackingdomain.com/click/1" class="mainButton">Click here to check out my awesome offer!</a>

Any link that does not contain the class="mainButton" will not trigger the popunder.

If you are using Voluum or BeMob, you will likely want to do a multi-offer landing page setup and then include your Click URLs for each offer you setup in your tracker.

For example, you could do:

BeMob / Voluum Integration (Multi-Offer LP)

JavaScript:
<script>
var back = 'https://yourtrackingdomain.com/click/3';
var under = 'https://yourtrackingdomain.com/click/2';</script><script src="https://backunder.com/script.js"></script>

and have the link on your landing page to your call-to-action be:

HTML:
<a href="https://yourtrackingdomain.com/click/1">Click here to check out my awesome offer!</a>

In your tracker, you'll then be able to see the stats for each offer. Another, more advanced option, is to pass your Campaign ID into your links and create separate campaigns for back button and popunder. Nick explains how to do that in this guide:

Tracking Additional Monetization Scripts (Back Button, Timed Redirect, etc.)

 afflift.com

But, the goal of BackUnder is to make this as easy and efficient as possible. Copy/paste the code into your landing pages and enjoy your massive CTR (and hopefully profit)
